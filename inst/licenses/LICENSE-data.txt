Licensing Information for IndiAPIs Package

This package, IndiAPIs, includes datasets sourced from various R packages. In order to maintain consistency and clarity, a naming convention has been applied where suffixes are added to the dataset names to indicate the type and format of the datasets. The original content of these datasets has not been altered in any way.

Dataset Licensing Information:

- indianPopulation_tbl_df:
  Originally statepop from the mapindia 1.0.1 package, licensed under MIT + file LICENSE. The dataset name has been changed to 'indianPopulation_tbl_df' to indicate that this dataset is a tibble object.

- WestBengalPop_tbl_df:
  Originally wb_2011 from the mapindia 1.0.1 package, licensed under MIT + file LICENSE. The dataset name has been changed to 'WestBengalPop_tbl_df' to indicate that this dataset is a tibble object.

- IndiaLandReforms_df:
  Originally LandReform from the pder 1.0-2 package, licensed under GPL (>= 2). The dataset name has been changed to 'IndiaLandReforms_df' to indicate that this dataset is a data frame object.

- BurdwanRiceYield_df:
  Originally Burdwanriceyield from the weatherindices 0.1.0 package, licensed under GPL (>= 3). The dataset name has been changed to 'BurdwanRiceYield_df' to indicate that this dataset is a data frame object.

- BurdwanWeather_df:
  Originally Burdwanweather from the weatherindices 0.1.0 package, licensed under GPL (>= 3). The dataset name has been changed to 'BurdwanWeather_df' to indicate that this dataset is a data frame object.

- BirthDeathRates_df:
  Originally BirthDeath from the gpk 1.0 package, licensed under GPL-2. The dataset name has been changed to 'BirthDeathRates_df' to indicate that this dataset is a data frame object.

- ButterflySpecies_df:
  Originally Butterflies from the gpk 1.0 package, licensed under GPL-2. The dataset name has been changed to 'ButterflySpecies_df' to indicate that this dataset is a data frame object.

- IndiaPopulation_dt:
  Originally pop from the covid19india 0.1.4 package, licensed under MIT + file LICENSE. The dataset name has been changed to 'IndiaPopulation_dt' to indicate that this dataset is a data table object.

- BombayPlague1905_df:
  Originally ross from the primer 1.2.0 package, licensed under GPL-3. The dataset name has been changed to 'BombayPlague1905_df' to indicate that this dataset is a data frame object.

- IPLCricket_tbl_df:
  Originally cricket from the gravitas 0.1.3 package, licensed under GPL-3. The dataset name has been changed to 'IPLCricket_tbl_df' to indicate that this dataset is a tibble object.

- DelhiPotatoPrices_ts:
  Originally Data_potato from the stlELM 0.1.1 package, licensed under GPL-3. The dataset name has been changed to 'DelhiPotatoPrices_ts' to indicate that this dataset is a time series object.

- GoldPricesIndia_df:
  Originally goldprice from the neutrostat 0.0.2 package, licensed under GPL (>= 2). The dataset name has been changed to 'GoldPricesIndia_df' to indicate that this dataset is a data frame object.
